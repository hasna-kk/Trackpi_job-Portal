# TrackPi Job Portal

## Overview

TrackPi Job Portal is a modern full-stack MERN web application collaboratively developed for **TrackPi Private Limited, Kochi**. The platform is designed to streamline recruitment, talent discovery, job management, and skill-based hiring workflows through an interactive and scalable web experience.

This project was built as a real-world recruitment management platform with separate frontend and backend architectures for scalability, maintainability, and deployment flexibility.

---

# Project Information

* **Project Name:** TrackPi Job Portal
* **Company:** TrackPi Private Limited
* **Location:** Kochi, Kerala, India
* **Architecture:** MERN Stack
* **Frontend:** React.js + Vite + Tailwind CSS
* **Backend:** Node.js + Express.js
* **Database:** MongoDB
* **Authentication:** JWT Authentication

# Installation & Setup

## Clone Repository

```bash
git clone <repository-url>
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

# Environment Variables

Create a `.env` file in the backend directory.

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_NAME=your_cloudinary_name
CLOUDINARY_KEY=your_cloudinary_key
CLOUDINARY_SECRET=your_cloudinary_secret
```
